Brief mathematical introduction to Perron-Frobenius theory

Consider the set of n×nn\times nn×n matrices MnM_nMn​, with complex entries. The algebraic structure (scalar multiplication by complex numbers, addition and multiplication of matrices) plays well with the operator norm, ∥A∥=sup⁡v∈Cn:∥v∥2=1∥Av∥2 \|A\| = \sup_{ v\in \mathbb{C}^n: \|v\|_2=1 } \|A v\|_2 ∥A∥=v∈Cn:∥v∥2​=1sup​∥Av∥2​ and the involution ∗*∗ (i.e. conjugate-transpose), making it a C∗^*∗-algebra1. A matrix A∈MnA\in M_nA∈Mn​ is positive semi-definite if it can be written as A=B∗BA= B^*BA=B∗B for some other matrix B∈MnB\in M_nB∈Mn​.

Let us now go one level higher: we will consider linear maps ϕ\phiϕ from MnM_nMn​ to MnM_nMn​, i.e. linear maps whose arguments are themselves matrices. Such a linear map is positive if it maps positive elements of MnM_nMn​ to positive elements of MnM_nMn​. Since ϕ\phiϕ is a linear map on a vector space (even if that vector space is itself the set of matrices), it has eigenvalues and eigenvectors. The eigenvalues are in general complex numbers, and number r=max⁡{∣λ∣:λ is an eigenvalue of ϕ} r = \max\{ |\lambda| : \lambda \text{ is an eigenvalue of }\phi \} r=max{∣λ∣:λ is an eigenvalue of ϕ} is the spectral radius of ϕ\phiϕ.
